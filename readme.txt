Displays the attachment's file size on that attachment's edit screen.

== Description ==

**The functionality of this plugin was merged into WordPress core in version 3.7.**

Adds a File Size field in the Save meta box on attachment edit screens. Can be helpful to identify large images/files.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
